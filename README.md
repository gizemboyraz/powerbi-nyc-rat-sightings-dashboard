# NYC Rat Sightings Power BI Dashboard

## Overview
This project is an interactive Power BI dashboard built using NYC Open Data to analyze rat sightings across different boroughs of New York City.

The goal is to identify trends, hotspots, and time-based patterns in reported sightings.

---

## Key Features

- Data cleaning and transformation using Power Query
- Data modeling (Star Schema approach)
- DAX measures for KPI calculations
- Interactive map visualization of sightings
- Time-based trend analysis (monthly/weekly)
- Borough-level comparison
- Drill-through pages for detailed analysis
- Dynamic filters and slicers

---

## Business Insights

- Identifies high-density rat sighting areas
- Shows seasonal trends in complaints
- Helps understand borough-level sanitation issues
- Supports data-driven urban planning insights

---

## Technologies Used

- Power BI
- Power Query (M)
- DAX
- Data Modeling
- NYC Open Data (Public Dataset)

---

## Project Purpose

This project was created as a portfolio piece to demonstrate my skills in Power BI, data modeling, and business intelligence reporting.
